# 🚀 How to Deploy DevFolio.in (FREE & NO CARD)

I have updated your code to be **Purely Client-Side**. This means you can host it for free on **Netlify** or **GitHub Pages** without needing a credit card for a server.

---

## ☁️ Deploying to Netlify (Fast & Free)

1.  **Download the ZIP**: Use the Gear icon at the bottom left to download your code.
2.  **Upload to GitHub**: Create a repository and upload all files.
3.  **Connect to Netlify**:
    *   Go to [Netlify](https://app.netlify.com).
    *   Add new site -> Import from GitHub.
4.  **Build Settings**:
    *   **Build Command**: `npm run build`
    *   **Publish Directory**: `dist`
5.  **Add Your Key (CRITICAL)**:
    *   Go to **Site Settings** -> **Environment variables**.
    *   Add a variable: 
        *   Key: `VITE_GEMINI_API_KEY`
        *   Value: (Paste your AIza... code here)
6.  **Deploy**: Your site will be live!
